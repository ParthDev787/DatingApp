# 💖 DatingApp

A full-stack dating application built using **ASP.NET Core** and **Angular** as part of my learning journey to become a professional full-stack developer.

---

## 🚀 Project Overview

DatingApp is a modern web application where users can:

* Create accounts and log in securely
* View and update their profiles
* Like and match with other users
* Chat in real-time

This project is built by following a real-world development approach with clean architecture and best practices.

---

## 🛠️ Tech Stack

### 🔹 Backend

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server

### 🔹 Frontend

* Angular
* TypeScript
* Bootstrap

---

## ✨ Features

* 🔐 User Authentication (JWT)
* 👤 User Profiles with Photos
* ❤️ Like / Match System
* 💬 Real-time Chat using SignalR
* 🔍 Search & Filter Users
* 📡 Online Presence Tracking
